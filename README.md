#  Uber Eats Style Menu
* SwiftUI 학습

## 기능 상세
* 메뉴바와 스크롤뷰가 함께 동작하는 기능 구현

## 학습
* @Binding

* ScrollViewReader
* onChange
* withAnimation

* @Namespace
* matchedGeometryEffect
(메뉴바가 순간이동하는 것을 자연스럽게 하기 위해서 )

* struct OffsetKet: PreferenceKey
* ViewModifier

* GeometryReader

* onPreferenceChange

* coordinateSpace
